# Configuração para criar um container com Tensorflow CPU

<br /><br />
## Para instalar o container
```shellscript
# docker-compose up --build
```


<br />
## Para iniciar o container
```shellscript
# docker-compose up
```


<br />
## Para rodar o Jupyter:
```shellscript
http://localhost:8888
```
